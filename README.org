** 开发板资源
*** MCU -  STMF407
- 168MHZ工作频率
- M4内核，支持单浮点
- 90nm工艺和ART加速，flash零等待
- 2个USB OTG，其中一个支持HS
- 专用音频PLL和2刚全工I2S
- 6个速度高达11.25Mb/s的USART
- 3个速度高达45Mb/s的SPI
- 3个IIC
- 2个CAN
- 一个SDIO
- 2个12位DAC
- 3个12位ADC
- 17个定时器，频率高达168MHz的16和32位定时器
- 512K Flash
- 192 + 4K Ram
